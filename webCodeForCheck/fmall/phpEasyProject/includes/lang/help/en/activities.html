Activity Manager

The Activity Manager is used for the entry of all hours that the staff members was take off project-positions within the project.

The acquired hours can be evaluated statistically, for example all hours within one month, or grouped by projects.

In the project-list, the user sees all projects where he/it is registered as project-participants for this project.

For the entry, you should select a certain project in the project-list. After it the list with appended project-positions will reload. Now select the project-position.

The current date is proposed the activity standard. You can write here own date. Alternatively, one can call a Pop-Up window with the dynamic calendar with the mouse-click on the calendar-symbol.

Now select the start of the activity with hours and minutes and afterwards the duration as well as activity end time. The minute-step (15, 30 or 60) is fixed from Administration (Direction) in the configuration of phpEasyProject for all staff members.

In the field “short-description” should you write the activity shortly and the field “description” is for detailed commentaries.

The Activity Management is no appointment book and primarily serves for the hour-entry. Therefore, the parallel activities are allowed not here. Start the next activity can immediately be prior the earliest with the end.

For editing an existing activity, one must click on this activity in the list with the mouse. It is reloaded after it in the editor-mask. After the alterations in the editor-mask, one must store changed as well as again registered activity with the mouse-click on disk-symbol.

Operating of the trash can-symbol deletes the selected activity irrevocably.

In the activity-list on the labeling-strip, the selected date stands with two arrows of both sides. The arrows serve for the folio for a day back as well as in front, and with the mouse-click on the date is inserted in the Pop-Up window of the calendars for faster date-jump.
