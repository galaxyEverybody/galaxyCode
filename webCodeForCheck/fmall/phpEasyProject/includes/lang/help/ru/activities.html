Учёт рабочего времени

Учёт рабочего времени служит для записей всего времени сотрудниками, по каждй позиции проекта, на которые это время было потрачено.

Записи по учёту рабочего времени можно впоследствии анализировать по времени (например, отработанные часы в течении месяца по каждому сотруднику), или группирую по проектам для выставления счетов по затраченному времени.

В списке проектов пользователи видят все проекты, в которых они непосредственно принимают участие. Участие в проектах управляется в разделе управления проектами, к которому имеют доступ менеджеры проектов и руководство компанией.

Для записи рабочего времени необходимо выбрать необходимый проект из списка. После этого система подгрузит автоматически список позиций проекта, одну из которых необходимо выбрать.

Для записи учёта времени система предлагает по умолчанию текущую дату. Для измениния этой даты введите в соответствующем поле требуемую или в качестве альтернативы щелчком мыши по иконке календаря рядом с полем даты откройте всплывающее окно динамического календаря.

Затем установите соответственно начало в часах и минутах и продолжительность (или оконочание - тогда продолжительность будет вычеслена автоматически). Шаг выбора минут в 15, 30 или 60 минут устанавливается руководством компании (администратором) в разделе настроек для всех один раз (и на всегда :).

В обязательном поле "краткое описание" напишите, чем Вы занимались (Помните, краткость - сестра таланта!). Для подробных описаний служит необязательное для записи поле "Подробности деятельности".

Учёт рабочего времени - это не календарь событий, но служит в первую очередь смыслу своего названия. Поэтому невозможность вести параллельные деятельности это не недостаток в программе, а специально созданное свойство. Начало последующей деятельности не может быть раньше окончания предыдущей

Для редактирования существующей записи щёлкните просто по полю в списке. После этого запись подзагрузится в маске редактирования. После внесения необходимых изменений не забудьте их сохранить щелчком по иконке дискеты.

Внимание! Нажатие иконки корзины удаляет выбранную запись без возможности последующего восстановления.

Для навигации в списке-ежедневнике используйте треугольные стрелки для листания вперёд/назад или вызов всплывающего календаря щелчком по дате.
