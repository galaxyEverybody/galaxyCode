Настройки пользователя

Каждый пользователь имеет возможность установить для листа учёта рабочего времени настройки начала и окончания рабочего дня (для более удобного показа - при занесении записи, выходящей за эти пределы, таблица автоматически увеличивается на требуемое значение).

Из доступного списка перевода программы выбирается нужный язык для конкретного пользователя.

Для удобства показа списка учёта рабочего времени возможно изменить шаг показа шкалы времени.
