Установка phpEasyProject

Требования

  • Веб-сервер или хостинг аккаунт с любой популярной ОС (Apache рекомендуется)
  • База данных (в настоящий момент только MySQL 3.23.x, 4.1.x или 5.x)
  • PHP (4.3.x or 5.x) с поддержкой MySQL Client Library и PHP class DOMDocument

Инсталляция

Если у Вас есть достаточные знания использования FTP и доступ к веб-серверу, то установка phpEasyProject может быть произведена следующим образом:

  1. Распакуйте архив phpEasyProject в локальной директории Вашей системы.
  2. Содержимое полученной папки перенесите на сервер.
  3. Установите права на запись на сервере для директорий /config и /templates_c для ВСЕХ (777 или -rwxrwxrwx)
  4. Запустите в Вашем броузере программу инсталляции по основному адресу phpEasyProject, где он находится на веб-сервере.
  5. Выполните инструкции по установке в пять шагов.
  6. После завершения установки удалите директорию /install.
  7. phpEasyProject готова к работе. Зарегистрируйтесь в phpEasyProject с помощью только что созданного аккаунта.
